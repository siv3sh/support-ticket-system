# Formatting Instructions for Word

**Before submitting, apply these settings in Microsoft Word:**

- **Font:** Times New Roman, 12 pt for body; **Chapter titles:** 16 pt Bold, CAPITALS, Centered; **Side headings:** 12 pt Bold, CAPITALS.
- **Line spacing:** 1.5.
- **Paper:** A4, one-sided.
- **Margins:** Left 1.5", Right 1", Top 1", Bottom 1".
- **Header (font 10):** Project name (left), Page number (right). Border line below header.
- **Footer (font 10):** "Department of Computer Science, CHRIST (Deemed to be University)". Border line above footer.
- **Page numbers:** Roman (iii, iv…) for front matter (Acknowledgments); no number for TOC/Title/Certificate; Arabic (1, 2…) from Chapter 1 (use Section Break).
- **Binding:** Light blue spiral.
- **Table of Contents:** Use Word’s automatic TOC; update before submission.
- **Figures/Tables:** Fig caption below, Table caption above; Times New Roman 10 pt.

Use **Heading 1** for chapter titles and **Heading 2** for side headings so the automatic TOC matches the structure below.
